
# Installation

## Main Environment
####  1. Install package
```bash
  npm install
```

####  2. Build package
```bash
  npm run build
```

####  3. Run app
```bash
  npm run start
```

####  4. Open application
- [Localhost:3000](http://localhost:3000)


## Test e2e Environment
####  1. Install package
```bash
  npm install
```

####  2. Install package
```bash
  npm run test
```
